# ByteHop

## Project Name
ByteHop

## Technologies
- Python
- Flask
- HTML / CSS
- SQLite (database)

## Description
ByteHop is an educational platform for students. 
It allows users to register, login, track their lessons, and earn certificates based on their progress.

## Features
- Home page with buttons to access:
  - Programming and AI course (textbook)
  - QuRo platform chapters
- Inside each lesson:
  - "Mark as Completed" button
- User account page:
  - Shows lessons completed by the user
- User authentication:
  - Register new user
  - Login
  - Logout
- Certificates and achievements page:
  - Shows a card for each course with percentage progress
  - If user reaches 100%, they are redirected to a congratulation page to see the name on the certificate

## How to Run
1. Install Python
2. Install required packages:
    pip install -r requirements.txt
3. Run the application:
    python app.py
4. Open your browser and go to the link shown in the terminal.
5. Use the registration/login page to start using the platform.
